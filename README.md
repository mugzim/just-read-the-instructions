# Впервые публикую сайт, используя GitHub Pages

## Прогресс:

**Давно** - купил забавный домен и хотел быстро опубликовать на нем заглушку. Оказалось, что не все так очевидно и стремительный порыв погас о стену отсутствия знаний/опыта. Оставил до лучших времен.

**14 января 2025** - сражался с настройками записей у регистратора, чтобы подружить мой домен с GitHub Pages. Потратил кучу времени, но результата не добился. В настроиках GitHub проверка DNS постоянно фейлилась и сайт не был доступен. В итоге, остановился на добавлении A-записи для апекс-домена и терпеливо дождался его появления. Сработало спустя несколько часов уже вечером. Бросился заполнять контентом.

**15 января 2025** - опубликована первая версия сайта: главная и гайд по оплате Tasa modelo 790.
